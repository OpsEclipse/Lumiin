# Component Installation Reference

Quick reference for adding shadcn/ui components.

---

## Install Command

```bash
npx shadcn@latest add [component] --yes
```

## Recommended to Add

### Forms & Inputs

```bash
npx shadcn@latest add form select checkbox radio-group switch textarea --yes
```

### Navigation

```bash
npx shadcn@latest add tabs navigation-menu breadcrumb --yes
```

### Overlays

```bash
npx shadcn@latest add dialog alert-dialog sheet drawer dropdown-menu popover tooltip --yes
```

### Data Display

```bash
npx shadcn@latest add avatar progress skeleton --yes
```

### Feedback

```bash
npx shadcn@latest add alert toast sonner --yes
```
